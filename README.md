# BPMN-Project
# 📌 Business Process Analysis & Redesign – Bottles Wine Club

## 📖 Project Overview
This project presents a **business process analysis and redesign** for *Bottles Wine Club*, focusing on identifying operational inefficiencies and proposing data-driven improvements using **Business Process Management (BPM)** methodologies.

The analysis evaluates the existing (“As-Is”) processes across key performance dimensions—**time, cost, quality, and flexibility**—to uncover bottlenecks such as:
- Delayed membership approvals  
- Inefficient order handling  
- Supplier-related delays  

## 🔄 Process Redesign (To-Be)
Based on these insights, the project proposes a redesigned (“To-Be”) process incorporating **Lean methodology principles** to eliminate waste, streamline workflows, and improve overall efficiency.

### ✅ Key Improvements
- Separating membership approval from order processing  
- Introducing parallel processing for credit checks and application assessment  
- Eliminating non-value-adding activities  
- Integrating supplier inventory systems for real-time visibility  

## 🎯 Outcomes
The proposed solution demonstrates how structured process redesign can:
- Reduce cycle time  
- Enhance customer experience  
- Support scalable business operations  

## 🛠️ Skills Demonstrated
- **Business Process Analysis** (BPMN, As-Is / To-Be modelling)  
- **Process optimisation** using Lean principles  
- **Stakeholder-focused solution design**  
- **Operational efficiency and workflow improvement**
